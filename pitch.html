Technical Architecture

Built on Google Cloud

AI Model Gemini Live 2.5 Flash — native audio, real-time vision, function calling
Agent Google ADK — 6 tools: render_panel, update_panel, clear_panel, show_timer, show_reminder, google_search
Frontend Vanilla JS, AudioWorklet (16kHz↑ 24kHz↓), WebSocket, Componentized Dynamic UI
Infra Cloud Run + Session Affinity — single container, auto-scaling, HTTPS

Key innovation: Generative UI — AI freely composes atomic components (steps, timers, buttons, text) into any interface.
